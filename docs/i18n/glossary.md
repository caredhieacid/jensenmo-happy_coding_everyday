# Translation Glossary

Keep product identifiers and lane names in English. Translate the explanation around them.

| Term | Intended meaning |
| --- | --- |
| HappyCoding Everyday | Product and skill display name; do not translate |
| Everyday | Default lane for one bounded outcome |
| Collaboration | Lane for independent partitions or valuable independent review |
| Delivery | Lane for durable coordination, high risk, or staged acceptance |
| dispatcher | The single workflow entrance that selects a lane |
| compact contract | Goal, scope, acceptance, and exclusions |
| fresh evidence | Evidence produced after the final relevant change |
| focused validation | The narrowest check that proves the requested behavior |
| blast radius | The systems, data, users, or releases that failure could affect |
| controlled writing | One accountable write line for shared files or contracts |
| read-only | No repository, external, or published state changes |
| irreversible action | An action whose impact is hard to recover or affects production/external state |
